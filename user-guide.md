# AcadCalc User Guide

## Welcome!
AcadCalc is a complete client-side Academic GPA, SGPA & CGPA Calculator.

## How to Use

### 1. Getting Started
- Open `index.html` in any modern browser.
- Toggle Dark/Light mode using the moon icon in the header.

### 2. Student Information
- Fill in your name and other details (auto-saved).

### 3. Select Grading System
- Choose your national grading system from the dropdown:
  - Bangladesh (5.00), Bangladesh (4.00), India (10.00), US (4.00), Pakistan, UK, ECTS, Japan, China, South Korea, Malaysia, or Custom.

### 4. Credit System
- **Equal Credit**: All courses have the same credit value.
- **Variable Credit**: Set different credits for each course.

### 5. Add Semesters & Courses
- Click **"Add New Semester"**.
- Add unlimited courses with Course Code, Name, Grade, and Credits.
- Calculations (SGPA/CGPA) update in real-time.

### 6. Charts & Summary
- Switch to **Charts** tab to see Grade Distribution (Pie) and GPA Trend (Line).

### 7. Conversion & Export
- Go to **Convert** tab to convert results between grading systems.
- Use **Export PDF** or browser Print for official-looking transcripts.

## Tips
- Everything is automatically saved in your browser (LocalStorage).
- Fully responsive on mobile.
- For best print results, disable Dark Mode before printing.
