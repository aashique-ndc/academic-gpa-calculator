# Developer Guide

## Architecture
- Modular Vanilla JS with ES6 modules
- Tailwind CSS for styling
- LocalStorage for persistence
- Chart.js for visualizations

## Adding New Grading System
1. Update gradingData.js
2. Add to select dropdown in app.js

## Extending Features
- Add new tab in index.html
- Create corresponding JS module

## Deployment
- Static hosting (GitHub Pages, Netlify, Vercel)

# Full Stack Structure
- index.html (Entry)
- assets/js/*.js (Modular)
- assets/css/*.css (Tailwind + Custom)
