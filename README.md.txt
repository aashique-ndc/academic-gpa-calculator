# MarsCal - Academic GPA, SGPA & CGPA Calculator

A modern, fully client-side academic calculator supporting multiple national grading systems.

## ✨ Features
- Bangladesh (5.0 & 4.0), India (10.0), US (4.0), Pakistan, UK, ECTS, Japan, China, South Korea, Malaysia
- Real-time SGPA/CGPA calculation
- Custom grading system creator
- Result conversion between systems
- Charts with Chart.js
- Print-ready official transcript
- PDF Export
- Dark/Light Mode
- LocalStorage persistence
- Mobile responsive

## 🚀 How to Run
1. Download or clone the repository
2. Open `index.html` in any modern browser
3. No server required
